---
title: 'Auto Messages'
description: 'Allows bleed to send repeating messages to a channel after a certain interval of time.'
---
<Warning> Timers are limited to **one** per channel and cannot be set at intervals for less than **ten** minutes. They must also be longer than **seven** characters in length. </Warning>

## Add a Timer
To add a repeating message using bleed, run the command `timer add #channel (time in minutes/m) (message)`. This can also be an embedded message, just add `{embed}` in the message field.

  <img src="https://i.imgur.com/gsxAoKR.png"/>
</Frame>
## Remove a Timer
To delete a set message using bleed, run the command `timer delete #channel `.
