---
title: 'Auto Messages'
description: 'Setup repeating messages to be posted in your server.'
---
<Warning> Timers are limited to 1 per channel, cannot be set at intervals for less than 10 minutes, and must be longer than 7 characters in length. </Warning>

## Add a timer
To add a repeating message using bleed, run the command `timer add #channel (time in minutes/m) (message)`. This can also be an embedded message, just add `{embed}` in the message field.

  <img src="https://i.imgur.com/gsxAoKR.png"/>
</Frame>
## Remove a timer
To delete a set message using bleed, run the command `timer delete #channel `.

## Related Commands

<AccordionGroup>
    <Accordion title="View your timer">
        Preview the timer you created.
        ```
        timer view #channel 
        ```
    </Accordion>
    
    <Accordion title="View all timers">
        Obtain a list of all timers created in the server.
        ```
        timer list
        ```
    </Accordion>
</AccordionGroup>
