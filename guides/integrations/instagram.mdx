## Instagram Stories Set Up
<Steps> 
<Step title= "Set up your desired feed">
```
instagram add (channel) (instagram handle)
```

<Frame type="glass" caption="">
  <img src="https://i.imgur.com/utfwbGO.png"/>
</Frame>
 
</Step>

<Step title= "Set a custom message for your feed">
This command lets you set your own custom message that sends with the instagram post. This message can be used to ping roles which is especially useful for alert pings.
```
instagram message (message)
```
<Info>To ensure bleed can ping roles, append the `--pingable on` flag at the end of your desired message</Info>

# Related Commands
<AccordionGroup>
    <Accordion title="Delete your feed">
        Delete an instagram feed you have made.
        ```
        instagram remove (channel) (instagram username)
        ```
    </Accordion>
    
    <Accordion title="View the current message set">
        View the feed message that has been set
        ```
        instagram message view (username)
        ```
    </Accordion>
    <Accordion title="Gather a list of the user feeds">
        Gather a list of all the current feeds that exist within the server.
        ```
        instagram list
        ```
    </Accordion>
</AccordionGroup>
