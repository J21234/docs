---
title: 'Booster Roles'
description: 'Booster roles enable your server boosters to make their own customizable role.'
---
<Steps>
  <Step title="Set the base role">
    Sets the role that booster roles appear below on the role list. This has to be set for booster roles to function.
    ``` 
    boosterrole base @role 
    ```
  </Step>
</Steps>

## Usage

<Steps>
    <Step>
        This is used to create your booster role.
        ```
        Syntax: boosterrole (color) <name>
        Example: boosterrole #ff0000 boss
        ```
    </Step>
    <Step>
        Renaming your booster role.
        ```
        Syntax: boosterrole rename <new_name>
        Example: boosterrole rename cool role
        ```
    </Step>
    <Step>
        Changing the color of your booster role.
        ```
        Syntax: boosterrole color <color>
        Example: boosterrole color #ff0000
        ```
    </Step>
    <Step>
        Removing your booster role.
        ```
        boosterrole remove
        ```
    </Step>
</Steps>

# Awarding a Specific Role upon Boosting
<Steps>
    <Step>
        Set the role that will be awarded to a user upon boosting.
        ```
        Syntax: boosterrole award @role
        Example: boosterrole award @immunity
        ```
    </Step>
    <Step>
        Remove the currently awarded role for boosting.
        ```
        Syntax: boosterrole award remove
        ```
    </Step>
    <Step>
        View the current role awarded for boosting.
        ```
        Syntax: boosterrole award view
        ```
    </Step>
</Steps>

## Related Commands

<AccordionGroup>
    <Accordion title="Randomize your booster role color">
        Randomize your booster role color.
        ```
        boosterrole random
        ```
    </Accordion>
    
    <Accordion title="Color your booster role the dominant color of your avatar">
        Set the booster role to be the dominant color of your avatar.
        ```
        boosterrole dominant
        ```
    </Accordion>

    <Accordion title="List all booster roles">
        List all booster roles in the server.
        ```
        boosterrole list
        ```
    </Accordion>
    <Accordion title="Cleaning up booster roles">
    Clean up unused booster roles if they weren't removed properly.
    ```
    boosterrole cleanup
    ```
    </Accordion>
</AccordionGroup>
